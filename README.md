# Journal Application

A Spring Boot-based journal application to create, manage, and analyze journal entries.

## Features
- User signup & login
- Add, edit, delete journal entries
- Sentiment analysis for entries
- Redis caching for performance
- Email notifications
- Spring Security for authentication
- MongoDB database

## Tech Stack
- Java 17
- Spring Boot
- MongoDB
- Redis
- Maven

## How to Run
1. Clone the repository
2. Configure your application.yml
3. Run `mvn spring-boot:run`
